02 — Typography

The Typography
System

A classical serif-based display font is paired with a clean, modern body font — communicating both spiritual depth and accessible warmth.

Cormorant Garamond
DISPLAY / HEADINGS — Weight 300–700 · Use for all H1, H2, H3 and pull quotes. Italics for emphasis and key phrases.
DM Sans — clean, warm, humanist sans-serif for all body text, labels, buttons, form fields, and navigation. Weights 300, 400, 500, 600.
BODY / UI — Weight 300–600 · High readability at small sizes. Never use below 12px.

15 — Responsive Design

Breakpoints &
Mobile-First Rules

The site must be pixel-perfect at all sizes. Design mobile-first — build for 375px and scale up. Test at every breakpoint listed below.

Breakpoint Label Key Layout Changes
320–479pxSmall MobileSingle column everything. Nav becomes drawer. Hero text 2.5rem. Stats stack 2×2. Footer stacks fully.
480–767pxMobileSingle column. Gallery 2-col. Programs horizontal scroll. Form full-width. Footer 2-col.
768–1023pxTablet2-column layouts. Gallery 3-col. Pillars 2×2. Founder section stacks. Footer 2-col.
1024–1279pxSmall DesktopFull 3–4 col layouts. Gallery 4-col. Hero full height. Nav full horizontal. Footer 4-col.
1280px+Large DesktopMax-width container 1280px centred. Generous white-space. Hero image fills viewport width.

Mobile Nav

Hamburger icon (top right). Tap opens a full-height slide-over drawer from the right with all nav links in large Cormorant Garamond text, Donate button at bottom, social icons row, and an X close button. Background: #061e15 with subtle teal gradient.

Touch Interactions

Hero slider: swipe left/right. Gallery: swipe through lightbox. Programs carousel: touch-scroll. Form inputs: proper mobile keyboards triggered (email, tel). Tap targets minimum 44×44px. Smooth momentum scrolling (-webkit-overflow-scrolling: touch).

16 — Motion Design

Animations &
Interactions

Motion should feel purposeful and organic — not flashy. It guides the user's eye and rewards exploration. All animations respect prefers-reduced-motion.

Page Load

Nav slides in from top (0.4s). Hero text stagger-fades up (0.2s delay per line). Stat strip slides up from bottom (0.6s). Overall: elegant, composed entrance — not aggressive.

Scroll Reveals

All sections: fade-up + translate-Y(30px) → (0) as they enter the viewport. Use IntersectionObserver. Stagger children by 80ms each for a cascading feel.

Hero Slideshow

5s per slide. Crossfade transition (1s). Subtle Ken Burns zoom (scale 1.0 → 1.06 over 5s). Dot indicators animate width on active state.

Counter Animation

Statistics count up from 0 to final value in 2s using easeOutCubic curve when the stat strip enters the viewport. Each stat starts with 200ms offset.

Card Hovers

Pillar cards: image zoom (scale 1.05) + border glow. Program cards: overlay slides up. Gallery images: teal scrim fades in + icon appears. All transitions: 0.3s ease.

Form Interactions

Field focus: teal bottom-border animates from left to right (transform scaleX). Button submit: loading spinner replaces text. Success: green checkmark animates in with scale bounce.
